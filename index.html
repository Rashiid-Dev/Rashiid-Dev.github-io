Item Catalog

This repo is a catalog website that has several features

* It has 3rd party login (Google Oauth).
* Logged in users can add/edit delete items that belong to them.
* They can see but not edit/delete items created by other users.
* It uses Python and the Flask library for the back-end.
* HTML5, CSS, Javascript for the front-end
* SQLite for storing user information.
* And a basic JSON API for other's to get data from the database.

Linux Configuration

To publish my Item Catalog website I choose AWS and a Linux Ubuntu virtual machine.
*It uses Apache2 as a webserver
